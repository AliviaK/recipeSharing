# Project Plan

### MVP
+ Create Account
+ Log in/log out
+ Search recipes by keyword
+ Browse all recipes
+ Create Party
+ Invite friends to Party
+ Edit Party details (Time, Recipe, Description, etc.)
+ Cancel Party
+ Upload recipe
+ Tag recipe by ingredient/dietary need
+ Delete recipe
+ Share recipe
+ Review recipe
+ Rate recipe
+ Comment on recipe
+ Upload photo
+ Delete photo
+ Scale recipe
+ Convert recipe to volume/weight
